费劲

说话费劲，相处费劲，想到和这个人有关的所有事都觉得费劲。

然后摆烂。

一盘玩烂的游戏当然只能重开啊。现实没有重开，只有886

一直觉得缘分已尽这个说法，像用尽力气拧干毛巾，手都被勒得发疼，肌肉也发酸打颤，然后你看毛巾，可能掉一滴水，可能什么动静也没有。

缘分已尽，就是我的手还在疼着呢，我万万不可能再放任自己不小心将它浸湿要再拧了。

长这么大，好好说过再见或是不欢而散的朋友，没有什么破镜能重圆的，因为尴尬。看到他们的脸就想起过去的煎熬心情，只会躲得越远越好。


#


就和人命一样，会回光返照。

缘分要尽了的时候，会有一段突然很温暖的时光。

#

你觉得你们之间有一条鸿沟，你特别着急特别用力地想要跨过去，那沟也不是多么多么的宽阔，但就是他不过来，你也过不去。

但是此时的你还是想着怎么能到对岸去。

但是忽然有一天，你意识到这个沟对你来说难以跨越，对他来说易如反掌，你意识到先前所做的一切都白费力气，你再也不想开口，而是一直一直的沉默。

你再也不想着怎么去跨过这条沟了，这条沟多宽多窄都跟你没关系了，你连着这条沟和对面那个人都不愿意花时间去想了，也不太记得清当初为什么一定要费尽心思去到河的对面了。


#
感觉到累。

是那种懒得再争吵，懒得再多说，甚至懒得再动情绪，深入骨髓的累。

如果还想争吵，还会生气，还会在意对方为什么这么想，想理解对方，也想让对方理解你，还会因为对方的某句话辗转反侧，那不是缘分已尽。

只有当对方不管说什么，你都不会再动气，你看着眼前这个人，说出很多曾经你觉得是戳心窝子的话，曾经能让你难受得睡不着的话，却不再有什么感觉。你只能感觉到一种乏味，一种灰暗，一种深入骨髓的疲倦，你闭上了嘴，甚至能笑一笑，你想：随便吧，那又怎么样呢。

眼前这个人怎么想的，如何说的，是个怎样的人，ta怎样看你，你都再也不介意了。你太累了，你挣扎过那样久，终于发现，也接受了这是一个死局，你沉默了。

这个时候，就是缘分已尽的时候了。


#

人有一种本能，叫“预感离别”。之前深刻体验过，当时就是两个人一起出去走路的时候，心里突然觉得：这可能是最后一次见面了。（于是就莫名地偷偷难过了一下，想着珍惜现在，现在能一起多待一分钟是一分钟…）果然，那是最后一次见面。后来，偶然间刷微博，才知道这个现象是“预感离别”，好多人也都有过这个感觉，所以有时候直觉这种东西还是挺准的，所以我觉得，当你能从心里（忽略外界的客观条件）预感到你们两个人的离别的时候，就是缘分尽了的时候吧


#
突然的开阔、寥落、豁达，仿佛一直罩在眼前的雾消散了，能看到更远的地方了。突然觉得以前那些事情一点都不重要，眼前这个人也不重要，再跟他多耽搁一分钟都会不耐烦；觉得自己以前斤斤计较的样子特别难看，觉得为了他掉眼泪、琢磨他的心情喜好、真情实感地为他操心、懂事地体谅、以他的工作为重、把自己的需要往后排……一点都不值得。醒悟真的是一瞬间的事儿，过了那个节点回头看以前的自己，跟看傻子一样。


#

我上完一天班身心疲惫，回到家打了一个小时游戏，又刷了一会儿抖音。

这样的日子过了有一个月。

现在我打开知乎再打开一盒牛奶，忽然看到这个问题，忽然想到原来我们已经一个月没有联系了。

我的生活没有你似乎也过得去，你的生活没有我似乎也向前走了。

不再有重要性，不再有很强的存在感，

不再有每日和每夜的牵扯，

不再有早安晚安，不再有今天吃什么，

不再有好烦还没下班。

没有你，却一切正常。

等我彻底忘记和你联系是一件重要的事情，

我知道我们缘分已尽。

啥是缘分呢？

就是我心里想着你，你也总想找我说话。

是画了三八线，又总是越界的课桌，

是买了两杯豆浆，你一杯我一杯，

是我就连洗澡的时候都要把手机放在浴室，

是我心情不好的时候，你百度了十几个段子发给我。

缘分啊，都是人为开始，也是人为结束。

#

我一直相信，有一种人，就是上天派来帮你渡劫的。这个人是你这段人生的光。

比如，我硕士时候的导师。

我肯定我前女友就是。

我前女友在得知我有一份base17万刀的工作之后的一个月和我提的分手。理由是觉得我的人生没有她也会很好。。。

之前我是穷逼博士，她愿意开车三个小时来看我，听我吹牛逼，一起吃挂逼面，追傻逼剧。

后来我得了新冠，她又来照顾我。

现在想想我还是不懂她为啥要和我分手。

明明都熬出来了。。。

恩，可能奥特曼拯救了地球就要回家了。

#

我认为缘分很大程度在于自己的念头，当你内心还对这个人有思念，并且很强烈的时候即使是分手状态那也是暂时，真正的缘分已尽是内心不会有太大的波澜起伏，平时也不太会想到这个人，或者说如果想到这个人也就一闪而过，没有急切的想要见面或者思念心切的想法，原来谈过一个对象，分手的时候很伤心，刚开始就总是念念不忘，甚至想再见他，有时候路过他的工作单位就想找个地方躲起来，偷偷看他是否从那里路过，后来一个月以后慢慢想开了，心里就不在伤心了，当他有一次约我吃饭的时候，我心里一点想见他的感觉都没了，只觉得去吃饭很浪费时间，所以所有的缘分都来自于内心的念头，当你内心已经不存在这种念头的时候就真的说明缘分已尽了，甚至觉得再去见面心里就不舒服，浪费时间，没意义等等这些感觉就是缘分已尽吧


#

有一天白天，我躺在床上，阳光照进房间。有一刻我觉得，自己好像不喜欢他了，可以离开他了，我要跑路了。就是那一刻。他也感觉到了。就是那一刻之后他突然对我好了，开始嘘寒问暖了，甜言蜜语了，甚至展望未来了。我们不再吵架了，和平相处了，甚至还挺甜。从片场很周折的捡道具做礼物送我，出差前抱着我说舍不得…在视频里会慈爱的看着我…上一次用这样的眼神看我，是三年前。他老了。我爱的男孩。从少年的青涩无邪高傲，到被社会扇太多次耳光的那种“成熟”沉稳…他看着我，我看着他。挂掉电话我一顿暴哭。我不再说话了，撒娇了，发脾气了，生气了。所有的和平，是分别夜的预演。


#

我曾经有过一个孩子，流产了，当时已经取名了。我把他埋在花盆里。搬家时，想最后带走花盆，结果我妈妈不知为什么打碎了所有的花盆，所有的土混在了一起。这是个倔强的孩子，他不想来我家，就连一钵土都不留给我。没有缘分就是这样，连一钵土都不会给你。但是我一直都记得他的名字，只在心里想起，不会再说出来。因为没有人能理解，那孩子哪怕对于他爸爸来说，都是不存在的，但是对于我来说，他实实在在的来过，在我肚子里呆过。

#

在彻底分手几个月前已经从各处各样的迹象，察觉到缘分快尽。尽管当时还能甜甜蜜蜜地拉手上街，还能相谈甚欢，还能拥抱接吻，说着永远在一起。可能真的是上天安排，那些存留在我们记忆中的珍贵的事物都在一个个消散。我们一起养的大金毛，出于不便只能转送给别人；我们一起去打卡过的彩虹斑马线，过了几天发现已经被拆掉消除；正打算去早就约定好的上海迪士尼，结果正好疫情突发；刚大吵完一架和好准备之后也好好在一起，突然知道之后都要异地；他一开始送我的定情项链，在某一天突然断了；甚至他说分手前的那个饭局，如果他不去参加，没听到一些话，我们或许不会分手，但是当时尽管犹豫，他还是选择了去。看吧，好像一切都在不可控制地冥冥之中地一点一点溃散。尽管表皮看起来完好无损，但内里已经早就垮得一塌糊涂。好像是这个世界都在阻拦我和他继续在一起。一开始我最早慢慢意识到这一点，还顽固地说我就要和上天对着干。结果是我遍体鳞伤。以后还是多信命。相信直觉。


#

一周见一次，对方来一次，我去一次。

每次周日晚上要分别的时候都很难，

直到有一次，我竟然更惦记着要早点去机场以免错过飞机。

简而言之，在某个时刻，有其他事情比对方更重要，或许就是缘分已尽吧。

#

两个人缘分尽了的时候，就会出现一个很明显的征兆就是互相之间开始不沟通了，这是因为互相之间都对对方失望了，知道沟通也改变不了什么，知道沟通也没有意义，于是他们开始选择不沟通，进入了一种破罐子破摔的情况。

#

分离是有预感的，会在那一段时间尽力满足对方的需求，无限包容对方，给予强烈的关怀和温柔。

会想把所有想一起做的事都做一遍，甚至已经开始预想规划了，感觉上要用尽所有的心力去坚持完成某个事情，有很深的执念。

但又有那么一瞬间觉得无望和没有意义。

#

关系快崩盘，但是双方还在维持的时候，如果你开始不停做关于对方的梦，那么过不了多久你们就彻底要分开了，并且永远不会和好。

我在快分别的时候都会不停做梦，过不了多久就真的再也没有联系了。

#

在彻底闹掰前，你们有段时间关系很好，无话不谈，双方互相分享，一度让你以为可以和这个人走下去。之后会突然出现某件事，让你们之间的信任出现裂痕，直到最后崩塌，你在也不想和对方分享任何事情，内心趋于缓和……


#

人这辈子，你会遇见谁，跟谁相遇，最后又跟谁在一起，其实都是注定好的。                                                     
当缘分走到了尽头，冷漠，敷衍，没有了信任，不想去沟通，不想去解释，就不要再挽留了，要离开的人，始终挽留不住。迷茫的纠缠，不过是延长散场的时间罢了，无济于事。                                                                                                            好好爱自己，善待自己，通透的人，总会活得很舒心，也会过得很幸福。


#

大学第一段恋爱的时候送了对方一条红绳脚链，他一条我一条，很便宜的那种，两元店买的。后来因为性情实在不合，消耗太大我提出了分手，但一直藕断丝连，知道不合适，但就是放不下，总是哭，体重也骤降，他也不断的联系我希望复合。

后来有一次学校组织班干部爬山，大部队爬的太猛，跟着爬到第二高峰的时候已经不行了，脸色煞白，接下来还要冲击第一高峰，实在不行了，遂让他们爬吧，我休息一会儿直接下山。

当时去的路上坐我旁边的隔壁班班长跟大部队爬上第一山顶之后，特意折返回来，拉着我一步一步走下去，当时十分感激。下一站去一个小镇玩儿，也玩的很开心。

那一天，我突然发现，世界这么大，阳光这么灿烂，有那么多善良的人，我为什么要为着一个不合适的人跟自己过不去。

返程后隔壁班班长把我送回去，我承认当时彼此其实心动了，当然最后并没发生什么，只是心动了一下而已，后续各自有各自的生活了。

神奇之处在于，我回到宿舍之后，前任发信息跟我说，他看到了我被别人送回去。我俩分手后很少偶遇，唯一一次被别人送回去被他撞见了，而且，脚链红绳莫名其妙断了。。。

他手忙脚乱接上了，后续也确实还联系了几次，但是，确实是那一天，我对他的所有情义都没了，甚至觉得以往的伤心莫名其妙，像中邪了一样。
